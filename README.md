# wizmatic
This is a tool designed to automate various actions within the game Wizard101.

# Installation notes
- GPU-NVIDIA (CUDA 12.8): `pip install -r requirements.txt`
- CPU-only: `pip install -r requirements-cpu.txt`

# Aspect Ratio Support/Requirements:
- 4:3   (800x600, 1024x768, 1152x864, 1280x960, 1440x1080, 1600x1200, 1920x1200)
- 16:9  (1280x720, 1600x900, 1920x1080, 2048x1152, 2560x1440)
- 43:18 (1720x720, 2924x1224, 3440x1440)
